DAB Comparator

Single-file browser tool for relative comparison of DAB intensity between two brightfield H-DAB images. Suitable for GitHub Pages. The main output is a bounded percent difference score for average DAB staining between the two images, reported on a 0% to 100% scale with direction shown separately.

Load two images by drag-and-drop or Browse. Then click Compare Images.

Method note: this browser version follows the same core logic as the Python script: tissue masking from brightness and saturation, H-DAB color deconvolution, and percent difference in average DAB staining. The main reported result is a DAB-only bounded percent difference score from 0% to 100%, with direction shown separately. It is still a relative comparison tool, not a validated clinical analysis platform.
